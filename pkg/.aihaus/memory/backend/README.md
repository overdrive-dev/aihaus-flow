# Backend Memory

This directory holds server-side knowledge accumulated from this repository.

Fresh installs intentionally start with no inherited backend facts.

## What Goes Here

- Migration conventions and ordering rules.
- API endpoint, serialization, and error-shape conventions.
- Integration-test and fixture patterns.
- Stack-specific gotchas confirmed in this project.

## What Does Not Go Here

- Cross-cutting project rules.
- Frontend-specific patterns.
- Reviewer output.
- Unanswered business questions.
